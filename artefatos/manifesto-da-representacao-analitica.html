Estrutura mínima recomendada

| Campo | Descrição | |---|---| | Identificador | Código único do artefato. | | Módulo associado | Módulo responsável por produzir ou consumir o artefato. | | Finalidade | Objetivo técnico-metodológico. | | Entradas | Dados, metadados, regras ou evidências necessárias. | | Saídas | Produtos, registros ou evidências geradas. | | Métricas associadas | Métricas usadas para avaliação. | | Responsável | Pessoa, equipe ou papel institucional. | | Versão | Versão do artefato. | | Limitações | Restrições, riscos ou incertezas. |

Critérios de validação

  • Possui finalidade explícita.
  • Possui estrutura mínima preenchida.
  • Está associado a fontes, ativos ou produtos rastreáveis.
  • Possui responsáveis e versão.
  • Pode ser utilizado pelos módulos posteriores.
